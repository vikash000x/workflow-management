## ğŸ“ Features

- **Multiple Workspaces:** Create and manage multiple workspaces, each with its own set of projects, tasks, and members.
- **Project Management:** Create, update, delete projects; assign team members to workspaces & projects.
- **Task Management:** Assign tasks, set due dates, track task status, and manage priorities.
- **Analytics Dashboard:** Track project progress, completion rate, task statistics, and team activity.
- **User Management:** Invite team members, manage user roles, remove users, and view engagement.
- **Real-time Sync (Webhook + Inngest):** Automated event-driven updates using Clerk + Inngest.
- **Modern UI/UX:** Fully responsive UI designed with Tailwind CSS and Lucide icons.
- **Secure Authentication:** Role-based access handled with Clerk (login, signup, org, membership).
